LambdaPi4Racket collection

Stephen Gaito < stephen@perceptisys.co.uk >

The LambdaPi4Racket collection provides the code to implement the LambdaPi4Racket interpreters. It currently consists of the STLCAlg and the lp4RacketLayer languages.

The STLCAlg language implements the acutal Algebraic Simply-Typed Lambda Calculus interpreter. While the lp4RacketLayer language provides strict control over which aspects of the underlying Racket/Scheme/Lisp programming language can be used by the overlying STLCAlg language.
