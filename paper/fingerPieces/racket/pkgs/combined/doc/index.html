diSimp collection

Stephen Gaito < stephen@perceptisys.co.uk >

The diSimp collection provides the code to implement the diSimplex interpreter. It consists of the diSimp and the diSimpRacketLayer languages.

The diSimp language implements the acutal diSimplex interpreter. While the diSimpRacketLayer language provides strict control over which aspects of the underlying Racket/Scheme/Lisp programming language can be used by the overlying diSimp language.
